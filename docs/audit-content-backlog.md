## Workflow

1. Pick one row per sprint (2-week cycle).
2. Before writing: search the repo and live site for an **existing** URL on the same intent; improve instead of fork.
3. Ship with internal links from homepage resources grid and one related blog post.
4. After publish: request indexing in Search Console; watch query + landing page pairing for 4 weeks.
